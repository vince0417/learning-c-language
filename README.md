# learning-c-language
Introdução a programação usando a linguagem C.

### Compilação e execução usando o GCC
```bash
gcc -o exec arquivo.c
./exec
```

### Fluxo básico com GIT
```bash
git add arquivo
git commit -m "O que foi feito"
git push
```

### Operadore lógicos
| Operador | Função
|----------|-------
| &&       | AND
| \|\|     | OR
